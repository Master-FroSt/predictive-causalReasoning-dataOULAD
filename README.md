Cara Menjalankan Project:
Buat sebuah folder bernama data dalam direktori kerja Anda.
Letakkan semua file .csv OULAD Anda (studentInfo.csv, studentVle.csv, vle.csv, studentAssessment.csv) ke dalam folder data/ tersebut.
Instal library yang diperlukan melalui terminal:
pip install -r requirements.txt
Jalankan file utamanya:
python main.py
Jika gagal:
python -m pip install --upgrade pip setuptools wheel
pip install numpy==1.26.4
